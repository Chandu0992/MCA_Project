Abstract

           “Message Authentication Code Using Improved Playfair Cipher” aims to design and implement new method for calculating MAC based on Playfair Cipher. Recent year people around the world are interested in sharing information among themselves. So integrity and security of the information while sharing is important. For this situation cryptography has been a boon to human kind which is intended to secure transmission of data. The objective of this project is to provide message integrity with security and authentication of the sender.
